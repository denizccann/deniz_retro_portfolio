# 💻 Y2K Cyber OS Portfolio 🌸✨

[![Live Demo](https://img.shields.io/badge/Live_Demo-Click_Here-ff66b2?style=for-the-badge)](https://denizccann.github.io/deniz_retro_portfolio/)

Welcome to my interactive, retro-themed desktop environment! This project serves as my professional software engineering portfolio, built entirely from scratch to showcase my skills in front-end development, DOM manipulation, and interactive web design. 

Instead of a standard scrolling webpage, I wanted to build an experience. Welcome to **Deniz_OS 2026 Edition**.

## ✨ Features

- **Draggable UI:** Fully interactive desktop icons and windows that can be dragged, maximized, and closed using pure JavaScript.
- **Retro Mini-Games:** Includes custom-built, playable versions of **Tetris** and **Snake** rendered on HTML5 Canvas.
- **In-Site Browser:** A working internet browser window utilizing the Bing Search engine.
- **Cyber Widgets:** - 🎵 A functional Winamp-style music player with a custom playlist.
  - 🔐 A real-time CCST-aligned Password Strength Analyzer.
  - 💻 A Matrix-style raining code terminal.
  - 🐾 An interactive digital Cyber Pet.
- **Dynamic Contact Form:** An AJAX-powered guestbook that sends emails directly to my inbox without requiring page reloads.
- **Aesthetic:** Custom SVG cursors, Y2K color palettes, GSAP background animations, and responsive retro UI components.

## 🛠️ Tech Stack

This project was built without heavy frameworks to demonstrate a strong understanding of core web technologies:

* **HTML5:** Semantic structuring and Canvas API.
* **CSS3:** Flexbox, Grid, custom keyframe animations, and retro styling.
* **Vanilla JavaScript (ES6+):** Event handling, OOP (Object-Oriented Programming) for games, DOM manipulation, and state management.
* **GSAP:** Lightweight library for smooth background entity animations.

## 📂 Project Structure

- `index.html`: The main desktop environment and window containers.
- `style.css`: All styling, animations, and custom cursors.
- `script.js`: The core logic for draggable elements, window management, games, and widgets.
- `assets/`: Contains all pixel art, background music, and profile images.
